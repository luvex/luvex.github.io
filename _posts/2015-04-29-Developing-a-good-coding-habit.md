Recently, President Obama called on every American to learn to code. Coding is becoming more and more important — it can be used not only in computer science, but also in finance, physics, and more. Although I am just a junior student majoring in software engineering, I have much experience in coding and I'd like to summarize how to develop a good coding habit from my perspective.
published: false

**First, be dedicated.** Whatever we do, we must concentrate on it. It is especially important in coding. When I first got into coding, I was interested in all programming languages and hoped to be proficient in all of them. At last, I got the opposite — I didn't do well in any of them. Then I began to concentrate on C++. After practicing for a long time, I found I understood programming languages much better, and it became much easier to pick up others.
published: false

**Second, practice and thinking are equally important.** Practice is very important — without tens of thousands of lines of code, nobody can become a great coder. However, thinking is also important. Only when we think deeply can we accumulate experience continuously. Without thinking, there will be little improvement even if practicing day and night.
published: false

**Third, learning to use tools is a good habit.** There is a proverb: a workman must sharpen his tools if he is to do his work well. GitHub, StackOverflow, and cloud storage are essential for an efficient coder:
published: false
- **GitHub**: deposit code and learn from open-source projects
- **StackOverflow**: ask questions and learn from others' experience
published: false
- **Cloud notes**: store reusable code snippets and access them anywhere

published: false
**Finally, confidence is also important while coding.** Every time I face a challenge, I try my best to solve it — searching the Internet or asking friends. Whichever approach I choose, I will learn new things that may benefit me in the future.

published: false
Coding is a fascinating thing. If we develop a good coding habit, I am sure coding will be much easier for us.
